# NYC Midnight 500-Word Fiction Challenge 2026 — Visualizer

Interactive visualization of the 1st-round assignments for the NYC Midnight 500-Word Fiction Challenge 2026.

**Live site:** https://tomgarske.github.io/nycm-2026/

## What's in it

- **Rounds & Rules** — three-round funnel (3,456 writers → 860 → 88 → 10), with the 48-hour mechanics
- **Economics** — $45 × 3,456 = ~$155K in fees; ~7.7% goes to prizes, 2% to charity, the rest to operations
- **Publishing Status** — random sample of 50 writers researched for prior publishing credits
- **Compete** — the three Garske family members (Tom / Benton / Valerie) and their respective groups, with tiered threat assessment
- **Prior Finalists** — 50 prior 500FC top-10 + HM finalists across 2023–2025
- **By Genre / By Object Theme / By Action Theme** — prompt clustering
- **Regions / Closest Rivals / All 86 Groups** — exploratory views of the 3,456-writer field

Pure static HTML + JS, no build step. Data was scraped from the public Airtable embed on `nycmidnight.com`.
